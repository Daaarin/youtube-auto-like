YouTube Auto Like

What's new :

  • Addon support black/white list

    • On a video, a new button is display in the addon option. This button allow to add the creator to the list. Once added you can removed it from the same place while watching one of his video.
    The manage button allows to read and edit the list.
    In white list mode all videos in the list will be liked. In black list mode, the videos in the list will never be liked.

  • Debug mode is present

    • Now the console is a less polluted by the addon. You can enable trace by activating the debug mode (how ? > see github).

  • Drop of the old youtube layout support

    • The 'new' material layout is out since 2015. 5y after, 98% of the users are using the material layout. Maintaining the old one is too time consuming for me regardless of my situation. You still can install the addon version 2.3.4 if you want to use the old youtube layout or create a PR on the github. Contributions are greatly appreciated.

Feel free to report any bug on Github.
